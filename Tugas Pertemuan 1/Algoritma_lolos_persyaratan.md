# || NOTASI ALGORITMA PSEUDO CODE ||

## ALGORITMA LOLOS PERSYARATAN

**Deklarasi Variabel**
 *string 	: nama*
 *int 		: umur, tinggi badan*

**Input**
 *nama*
 *umur*
 *tinggi badan*

**Kondisi**
 *If*

 *umur >= 18*
 *tinggi badan >= 169*
 *keterangan :="Lolos"*

 *Else*	

 *keterangan :="Tidak lolos"*

**Output**
 *nama*
 *keterangan*
